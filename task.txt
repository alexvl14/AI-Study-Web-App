You are a highly skilled Senior Frontend Engineer specializing in creating polished, user-focused, single-page React applications using Tailwind CSS. We are building a modern web application for university students focused entirely on studying, inspired by NotebookLM but customized for intensive active recall. The design must be clean, minimal, and use a calming but professional color palette (e.g., shades of gray, white, and subtle, deep blue accents).

Page 1: The Notebook Dashboard
Design a dashboard page where users select or create a notebook.

Header: A simple navbar with the app logo ("StudyLM") and a user profile avatar.

Existing Notebooks: Create a section that displays a responsive grid of "notebook cards." Each card should show:

A clean notebook icon.

The notebook title (e.g., "History of Sofia, 19th Century").

The number of uploaded documents within it.

A "last modified" timestamp.

The "Add New" Feature: The first card in the grid should be a special "Create New Notebook" card. It must be marked distinctly (e.g., a simple dashed border with a large "+" icon and centered text). When clicked, it should trigger a modern modal to input a new notebook name.

Page 2: The Main Notebook Workspace
When a notebook is selected, the view must switch to the main workspace. This page must use a sophisticated 3-column layout.

Column 1: Materials & Sources (Left Panel - ~25% width)

A sidebar layout with a clean header titled "Sources."

Below the header, a prominent, modern "Upload Documents" button (PDF/TXT icon).

The main area should list uploaded documents in a clean vertical list (similar to NotebookLM). Each item needs:

A file type icon.

The filename.

A small checkmark or icon indicating if it has been indexed for AI.

This column must have internal scrolling.

Column 2: Gemini Interaction (Middle Panel - ~45% width)

This is the main chat interface. It must have a central chat box area with proper scroll behavior (new messages appear at the bottom).

Use standard chat bubbles: a lighter gray for user input and the accent blue for Gemini's response (or vice versa, the key is distinction).

At the very bottom, there must be a modern, prominent text input area with a subtle send icon.

Column 3: Dynamic Study Plan (Right Panel - ~30% width)

A sidebar titled "AI Generated Study Plan."

This panel must display a list of key topics that Gemini has generated based on the conversation and the uploaded docs.

Topic Markers: The generated topics must be stylized distinctly. Use shades of gray for unvisited/inactive topics, and a subtle blue for the current/active topic to clearly indicate progress and focus. Make this look like a clear timeline or checklist.

Component: Interactive Topic Modal (Core Logic)

When a user clicks on any topic in the Study Plan list, it must open a professional, modern "Drawer Modal" that slides in from the right.

The modal content must handle a large amount of text gracefully, ensuring perfect scrolling and fixed headers. The content of this modal is:

Fixed Header: Topic Title.

The Content Section: Displays all the extracted text for that specific topic in a highly readable font.

The AI Reading Button: In the sticky header or footer of this modal, there must be a prominent button with a speaker/sound icon that says "Turn on AI Reading" (a toggle button).

The Closing Action: At the very end of the text (requiring the user to scroll down), there must be a distinct "Quiz Me" section. This must show 2-3 short, multiple-choice questions about the text above, followed by a "Check My Answers" button.

<article class="prose prose-slate max-w-none">
<p class="text-lg leading-relaxed text-on-surface/90 font-medium mb-8">
                        Photosynthesis is often regarded as a purely classical chemical process. However, recent groundbreaking research in quantum biology suggests that <span class="bg-primary-fixed px-1 rounded">quantum coherence</span> plays a pivotal role in the efficiency of energy transfer within plants.
                    </p>
<div class="bg-surface-container-low p-8 rounded-2xl mb-10 border-l-4 border-primary">
<h4 class="text-sm font-extrabold uppercase tracking-widest text-primary mb-4">Key Synthesis</h4>
<p class="text-on-surface-variant leading-relaxed">
                            Excitons (energy packets) don't simply "hop" from one pigment molecule to another. Instead, they behave like waves, exploring multiple paths simultaneously to find the most efficient route to the reaction center. This phenomenon, known as the <span class="italic font-semibold">Quantum Random Walk</span>, prevents energy loss as heat.
                        </p>
</div>
<h3 class="text-xl font-bold text-on-surface mb-4">Mechanism of Excitation</h3>
<p class="text-on-surface-variant leading-relaxed mb-6">
                        When a photon strikes a chlorophyll molecule, it creates an exciton. In a traditional classical model, this exciton would move randomly, akin to a drunkard's walk. However, in the Fenna-Matthews-Olson (FMO) complex, we observe long-lived quantum beats. These beats are evidence that the system is in a superposition of states, allowing for near 100% efficiency.
                    </p>
